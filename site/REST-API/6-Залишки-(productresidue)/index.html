    Залишки товарів (productresidue)

    Контролер productresidue надає доступ до поточних залишків номенклатури по складах, цінах та серійних номерах. Дані залишків зберігаються in-memory на сервері.

    Ендпоінти

    Метод URL Призначення
    POST /productresidue/search/ Пошук залишків за текстом, фільтром, сторінкою
    POST /productresidue/objectlabels/ Мітки полів та перелік складів
    POST /productresidue/getinfo/ Розширена інформація по конкретній номенклатурі
    POST /productresidue/analoggroup/ Залишки аналогів номенклатури

    POST /productresidue/search/

    Пошук залишків з підтримкою пагінації, фільтрації по номенклатурі та типах цін.

    Параметри запиту

    {
    "q": "кава",
    "pa": null,
    "pl": ["guid-типу-ціни-1", "guid-типу-ціни-2"],
    "isn": false,
    "nfilter": ["guid-номенклатури-1"],
    "pi": 0
}
    Параметр JSON-ключ Тип Опис
    Пошуковий рядок q string Текстовий пошук по найменуванню, штрихкоду, артикулу
    Батьківська група pa GUID Обмежити пошук у межах групи номенклатури
    Типи цін pl array of GUID Перелік GUID типів цін для включення у відповідь. Якщо не вказано — ціни не повертаються
    Серійні номери isn bool true — включити залишки в розрізі серійних номерів
    Фільтр номенклатури nfilter array of GUID Повернути залишки тільки для вказаних позицій номенклатури
    Сторінка pi int Індекс сторінки (з 0). Розмір сторінки визначається сервером

    Відповідь

    {
    "data": [
        {
            "ic": "00003",
            "n": {
                "props": { "price": 456.00, "cu": "грн", "bu": "шт" },
                "oType": 2,
                "f": 8,
                "ic": "00003",
                "c": "Назва номенклатури",
                "id": "eb845c787af741569d0ac66dc517afd9"
            },
            "ar": "ART-001",
            "fsc": "1234567890128",
            "qsku": {
                "Офіс (Власник)": { "nr": -30.0, "nrz": 1.0, "nw": -5.0 },
                "Магазин":        { "nr":   0.0, "nrz": 0.0, "nw":  5.0 },
                "Вопак":          { "nr": -18.0, "nrz": 0.0, "nw": 16.0 }
            },
            "pprl": {
                "guid-типу-ціни": {
                    "cu":  { "id": "guid", "c": "UAH" },
                    "pr":  250.00,
                    "rp":  220.00,
                    "rc":  { "id": "guid", "c": "UAH" }
                }
            }
        }
    ],
    "advData": { "pages": 3 },
    "message": null,
    "sw": 12
}

    Поля елемента відповіді (product_residue)

    Поле JSON-ключ Опис
    Код ic Внутрішній код номенклатури
    Номенклатура n Об'єкт номенклатури (детально нижче)
    Артикул ar Артикул. Відсутній якщо пустий
    Штрихкод fsc Основний штрихкод. Відсутній якщо не заповнений
    Залишки по складах qsku Словник: ключ — назва складу, значення — об'єкт залишку
    Ціни pprl Словник: ключ — GUID типу ціни, значення — об'єкт ціни. Присутній тільки якщо передано pl
    Залишки по серіях qsn Присутній тільки при isn: true

    Об'єкт номенклатури (n)

    {
    "props": { "price": 456.00, "cu": "грн", "bu": "шт", "img": "2/guid" },
    "oType": 2,
    "f": 0,
    "ic": "00003",
    "c": "Назва номенклатури",
    "id": "eb845c787af741569d0ac66dc517afd9"
}
    Поле Опис
    id internalGuid без дефісів
    c Найменування
    ic Код
    f Flags
    props.bu Базова одиниця виміру
    props.price Ціна (з першого доступного типу ціни)
    props.cu Валюта ціни
    props.img Версія та GUID зображення (якщо є)

    Структура залишку (qsku[назва складу])

    Поле JSON-ключ Опис
    Залишок nr Поточний залишок
    Резерв nrz Зарезервована кількість
    Очікується nw Кількість в очікуванні надходження

    Структура ціни (pprl[guid])

    Поле JSON-ключ Опис
    Валюта cu Валюта ціни {id, c}
    Базова валюта bcu Базова валюта (якщо відрізняється від cu)
    Ціна pr Ціна продажу
    Ціна акції rp Акційна ціна (якщо є)
    Валюта акції rc Валюта акційної ціни

    Пагінація

    advData.pages — загальна кількість сторінок. Для отримання наступних сторінок збільшувати pi:

    { "q": "кава", "pi": 1 }
{ "q": "кава", "pi": 2 }

    POST /productresidue/objectlabels/

    Повертає мітки всіх полів відповіді та перелік складів. Використовується для побудови заголовків таблиці залишків.

    Запит

    Тіло порожнє або {}.

    Відповідь

    {
    "data": {
        "ic":  "Код",
        "n":   "Товар",
        "fsc": "Штрихкод",
        "fsn": "Серійний номер за замовчуванням",
        "ar":  "Артикул",
        "qsku": {
            "bd71a951150d4cc9a800b9963302d08e": "Офіс (Власник)",
            "dda27befbf2d4d47a19c4d7adea26ecb": "Магазин",
            "8efcb32d1db54f5e9c628a85eea9664d": "Вопак"
        },
        "qskuav": {
            "bd71a951150d4cc9a800b9963302d08e": { "av": true },
            "dda27befbf2d4d47a19c4d7adea26ecb": { "av": true }
        },
        "qsn":  "Залишки по серіям",
        "pprl": "Всі ціни"
    },
    "sw": 0
}

    qsku — словник: ключ — GUID складу без дефісів, значення — назва складу. Саме ця назва використовується як ключ в qsku відповіді /search/.

    POST /productresidue/getinfo/

    Розширена інформація по конкретній позиції номенклатури: зображення, властивості, залишки серійних номерів.

    Параметри запиту

    {
    "n": "guid-номенклатури",
    "pl": ["guid-типу-ціни"]
}

    Відповідь

    {
    "images": [ /* список зображень */ ],
    "properties": [ /* список властивостей */ ],
    "serialnumberlist": [ /* залишки по серійних номерах */ ]
}

    Поля images, properties, serialnumberlist присутні тільки якщо містять дані (інакше null).

    POST /productresidue/analoggroup/

    Повертає залишки для всіх аналогів номенклатури (по групі аналогів).

    Параметр q містить JSON-рядок з GUID номенклатури:

    {
    "q": "{\"n\":\"guid-номенклатури\"}",
    "pl": ["guid-типу-ціни"],
    "pi": 0
}

    Відповідь аналогічна /search/.
